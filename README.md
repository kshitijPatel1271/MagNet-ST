# MagNet-ST
MAGNet-ST: A Multi-Scale Adaptive Graph Network for spatial-temporal traffic forecasting using GCN, TCN, and cross-attention-based context-aware feature fusion on the METR-LA dataset.
This project uses the METR-LA dataset for traffic speed forecasting.

The dataset can be downloaded from:
https://github.com/liyaguang/DCRNN

Download the dataset and update the data path in the notebook before running the experiments.

## Experimental Setup

Experiments were conducted using Google Colab with an NVIDIA T4 GPU.

## Reproducibility Note

The notebook was developed and tested in Google Colab using GPU acceleration. Training time and memory requirements may vary depending on the available hardware.
